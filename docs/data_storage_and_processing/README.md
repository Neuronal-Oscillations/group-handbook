# A howto for using ARC and RFS
## Purpose
We here explain the steps of using the ARC and RFS. This includes getting, accounts, loggin in to the systems and executing MNE Python scripts. 

## Prerequisites
- A SSO (Single Sign-On) account at the University of Oxford
- VPN access to the University of Oxford network (required for off-site access)

  
## Steps

- Accessing [ARC](https://github.com/Neuronal-Oscillations/group-handbook/edit/main/docs/data_storage_and_processing/accessing_ARC.md)
- Getting storage on RFS: [Storage of OHBA imaging data](https://github.com/Neuronal-Oscillations/group-handbook/blob/main/docs/data_storage_and_processing/storage.md)
