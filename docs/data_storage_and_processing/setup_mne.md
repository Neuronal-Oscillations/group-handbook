# First things first

Follow the instructions in group-handbook/docs/data_storage_and_processing/first_steps.md to set up your environment correctly.

# Install MNE

```bash
cd $DATA/git-repos
bash group-handbook/code/hpc/setup_mne.sh
```

This will download FLUX, install a new MNE environment in `$DATA/environments`and also install a jupyter kernel.
