# Group Handbook    

This the handbook of the Neuronal Oscillations group addressing technical implementations. The aim is not to replicate existing documentation but rather to provide pathways for how-to specific to the group.

If you have ideas for improvements or new sections lets us know. The best approach for forwarding ideas is to use 'Issues' in the task bar of GitHub.
