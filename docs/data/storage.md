# Storage of OHBA imaging data
## Purpose
Recommendation for storing of OHBA imaging data incluyding MEG, OPM and MRI data. We propose of use the Oxford Research File Service (RFS) for  anonymised data (e.g. raw OPM data with no identiers).
OneDrive can be used for non-anonymised data. 

## Prerequisites
A SSO account at University of Oxford

## Steps

### Requestions a RFS storage 
- Follow steps on https://www.it.ox.ac.uk/article/oxford-research-file-service-rfs-available-for-storing-your-live-research-data
